# BHIT Platform / SiteFlow Pro

This repository contains the staging version of the BHIT Platform along with a
specification for **SiteFlow Pro**, a modular construction and installation
project management application.

The spec is defined in [`appSpec.js`](appSpec.js) and outlines the key
features, roles, advanced options, and tech stack for building the platform.

## Contents

- `index.html` – basic placeholder page
- `platform/` – UI placeholder with navigation
- `appSpec.js` – JavaScript file exporting the full application specification

## Deploying

This repo can be pushed to GitHub to share the current staging site and the
associated spec. No build step is required for these static files.
